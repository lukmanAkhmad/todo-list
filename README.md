# webpack-template

Visit Website: [Todo List App](https://lukmanakhmad.github.io/todo-list/)

## Features

- View all lists.
- View all tasks in each list
- Expand a single task to see/edit its details.
- Delete a list.
- Delete a task.
- Add due date.
- Add priority.
- Checklist a task.
- Edit lists.
